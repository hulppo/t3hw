Documentation

Class Connection. This is an abstract class from where a real connection should inherit. The inheriting connection specifies fully if it is a command-line, TCP/IP or a dummy connection. Connection uses a pointer to the Server class. Connection is started by ConnectionHandler as a thread.
Connection(long connId, std::string userName, Server* serv)
Constructor. This is the constructor for Connection, every inheriting class should call the constructor of this class before making their own initializations.

Parameters:
connId - Identification of type long. Should be unique for every Connection object.
userName - Name of the user, type std::string
serv - Pointer to the Server, type Server*

Destructor

virtual void sendMsg(Message* msg)
Sends Message* to user. Used for outgoing messages. This method gets called when there is a message to send to the user of this connection. It should be overwritten in the inheriting class.

Parameters:
msg - Type Message*

virtual void sendMsg(PrivateMessage* pMsg)
Sends PrivateMessage* to user. Used for outgoing messages. Should implement the sending of a private message. Can be overwritten in the inheriting class, otherwise calls sendMsg(Message*).

Parameters:
pMsg - Type PrivateMessage*

virtual void sendMsg(ChannelMessage* cMsg)
Sends ChannelMessage* to user. Used for outgoing messages. Should implement the sending of a channel message. Can be overwritten in the inheriting class otherwise calls sendMsg(Message*).
